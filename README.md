# 🚀 AI PPT Generator

AI-powered web app that generates professional PowerPoint presentations from a simple text prompt using Groq and Python.

---

## ✨ Features

* 🧠 Generate slide content using AI (Groq LLM)
* 📊 Automatically creates PowerPoint (.pptx) files
* 🎯 Structured slides with headings, bullets & speaker notes
* 🎨 Clean and modern slide design
* ⚡ Fast and easy to use

---

## 🛠️ Tech Stack

* Python
* Flask
* Groq API (LLM)
* python-pptx

## ⚙️ Setup Instructions

### 1. Clone the repository

```
git clone https://github.com/thakkarmeet909/Slide-ai.git
cd Slide-ai
```

---

### 2. Install dependencies

```
pip install -r requirements.txt
```

---

### 3. Add environment variables

Create a `.env` file and add:

```
GROQ_API_KEY=your_api_key_here
```

---

### 4. Run the project

```
python main.py
```
